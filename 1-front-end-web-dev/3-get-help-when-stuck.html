3 - How to get help when stuck?

Notes:

Expectation v reality

  • What did you expect your code to do and what actually happened?
  • Then try to work out the difference

Errors

  • When you get error messages, search for them on Google if you don't know what caused them
  • Can also rewatch video from just before point started encountering problems
  • Check your code against hers. Completed code at end of section you can download
  • Q+A for every lesson on Udemy to see if anyone with same issues (or ask if not)
  • Ask in Q+A by saying what I expected, what happened, screenshot of code & debug console
