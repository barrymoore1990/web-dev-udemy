1 - How does the internet and websites actually work?

Notes:

How does the internet work?

  • Every computer connected by wires to the internet
  • Computer (client) to servers (serving you), servers on 24/7
  • Type in website, connects to your ISP (internet service provider e.g. virgin)
  • ISP sends that do DNS (Domain Name Server)
  • DNS will look up the IP address for that server
  • Every computer connected to internet has it's own IP, so each can be found and connected
  • DNS then sends that back to ISP to client(you)

How do websites work?

  • Browsers are pieces of software which allow you to lookup ip addresses of websites
  • Websites consist of HTML, CSS, JS
  • HTML is the builder of a house, put up walls, the structure
  • CSS are for styling. Painters and decoraters
  • JS allows you to do things. Electrician to turn on the lights, plumber allows toilet to flush

Keywords/Questions:

  • ISP - Internet Service Provider
  • Every computer has it's own IP

Summary:

Every computer connected by wires, which have their own IP addreses, like a postal address, so you can send things to each other.
