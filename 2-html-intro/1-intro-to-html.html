Intro

HTML tags

  • Heading tags need opening and closing tags
  • Line break (br) are self closing
  • To see self closing tags can see on devdocs info about element, tag omission

Anatomy of HTML tag

  • Start tag, content, end tag
  • Self closing tag e.g. br
  • hr (horizontal line) - hr (HTML element) size="3" (HTML attribute)
  • Attributes give more info to browser to specify modifications
  • In devdocs, you can see attributes for each element
  • Comment tags in HTML are done like so
