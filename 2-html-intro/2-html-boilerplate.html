2 - HTML Boilerplate

Notes:

Intro

  • Boilerplates are the template of HTML code

Head element

  • Head element is a container for metadata (data about data)
  • Contains style, title, meta, link, script and base elements
  • utf-8 is default character set for us, but can be others for different languages, e.g Greek and Arabic

Text structure

  • Indents are used to show what hierarchial level they are (child / parent)
  • Good practice to use em not i for italic. HTML is more about structure
  • Use strong not bold

Lists

  • ul for bullet points, ol for ordered (numbers, letters, etc)
  • Ordered lists can be started not at 1 or a by giving it an attribute, e.g. start="8"

Image elements

  • img are self-closing but need more than just img, needs a source (src)
  • Img's can be local or link
  • Alt is alternative text

Links & Anchors

  • href is hyperlink reference

Keywords/Questions:

  • Head element is a container for metadata
  • use em and strong not i and b

Summary:

Boilerplates are templates of html code, and utf-8 is the default character set to be used in meta tag.
