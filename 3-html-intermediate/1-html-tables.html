1 - HTML Tables

Notes:

Structure

  • Table tag to open and close
  • tr for each table row
  • td for each cell (table data)
  • Also have their own header (thead + th), body (tbody) and footer (tfoot)
  • Only need to worry about rows & cells, not columns
  • mdn shows a lot of table attributes. However, have thumbs down because they're deprecated - don't recommend using, old-fashioned (use css)
  • Can use HTML tables for layout (but not recommended)
