2 - HTML Forms

Notes:

Structure

  • Won't see their full functionality until we get to JS
  • Label for the text above a cell
  • Input for the form cell
  • Submit button is also an input type (huge number of different input types)
  • Input self-closing tag, where as label is opening & closing
  • Textarea for big text input like your message - rows show how high and columns how wide the text area should be
  • Form action is what should happen when you click submit
  • enc type is encoding type of data - text/plain means just plain text
  • Giving a name to each attribute helps identify the data



Keywords/Questions:

  • form action is what happens after click
  • Labels and input key parts

Summary:

Won't see full functionality until JS. Can process user inputs and do things with that data
