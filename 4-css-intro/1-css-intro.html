1 - Introduction to CSS

Notes:

Intro

  • Original websites just used HTML & was impossible to look good. The best looking ones were really complicated code
  • Came up with CSS which is useless by itself

Inline CSS

  • Can do css on html file by using style
  • To do this, the element, e.g. background-color, it's attribute and ; must be enclosed in quotations
  • Random note - colorhunt.co really good site for finding colours
  • Gets error prone doing this method - e.g. if changing colour of 4 horizontal lines, got to remember to do every single 1 rather than just once

Internal CSS

  • Put style elements in head of html
  • Worth doing something dramatic to start with to make sure it's error free - not just css but other things too
  • CSS have default values - hr has quite a few
  • Everything that exists on webpages are essentially just boxes - use css to change any of them
  • Chrome plugin called Pesticide shows all of the boxes on a page
  • Internal doesn't really get used because it has to be copied into every page. That's why you use external css

Px values - margins, borders etc

  • Not 100% but from what I can make out this applies to all where you can have px size for top, right, bottom, left
  • 1 value means apply to all
  • 2 values means apply to 1. Top + bottom. 2. Right + left
  • 3 values mean apply to 1. Top. 2. Right + left. 3. Bottom
  • 4 is top, right, bottom, left, clockwise order
