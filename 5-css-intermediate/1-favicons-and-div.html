1 - Favicons and Divs

Notes:

Favicons

  • Favourite Icons - users only used to see them when they bookmarked a link (favourited it)
  • Can use favicon.cc to create one easily
  • Upload by doing link rel="icon"

Div

  • Div's allow us to divide the site up
  • Can stucture each div differently
  • Div's don't do anything unless you use CSS with them

Keywords/Questions:

  • Link rel="icon"
  • Divs used to divide site up
  • Divs don't do anything unless combined with css
