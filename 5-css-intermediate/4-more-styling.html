4 - More Styling

Notes:

Font Styling

  • Designers start off web design by picking colours and fonts
  • 2 main font families, serif (feet at end of letters) & sans-serif
  • Normally default to serif(times), sans-serif default(arial)
  • Use font-family to define
  • Web safe fonts which most operating systems have
  • Use cssfontstack.com to see %'s of people who have certain fonts. Click on font you like on here & it can provide fallbacks similar
  • Can do font embedding to ensure everyone sees font how you want
  • Google Fonts have a lot of royalty free fonts
  • Google Fonts provide link for you to copy and paste into header of your site - if users don't have the fonts, it'll install them

Sizing

  • If you use a font size & user changes font to e.g. large or small on their browser, this one won't change but others might because it's not dynamic
  • Can use 100% (equivalent to 16px) - if you want 48px normal then it would be 300%
  • Em is also dynamic (completely different for em for emphasis / bold
  • 1 Em is also 16px
  • Em & %'s get inherited from the parent - so if 2 in body then 5 in h1, it will be 7
  • In css 3 can use rem (root em) which doesn't add parent font size. Rem is recommended to use
  • Can use rem for margins, padding, etc too but normally prefer not to but just a preference

Float and Clear

  • Use float to wrap elements next to each other
  • If you don't want another element to wrap around a float element, use clear (on e.g. paragraph element)
  • Float one of most used & abused elements - recommend to use it only when really necessary & only for what its meant
  • Purpose is to wrap text around an element & don't use it for positioning
  • Instead, use relative, absolute, margin, padding etc for layout

Keywords/Questions:

  • 2 main font families, serif have feet, sans-serif don't
  • Use font embedding & cssfontstack
  • Use rem for font-size
  • Use float & clear for wrapping, not for layout
