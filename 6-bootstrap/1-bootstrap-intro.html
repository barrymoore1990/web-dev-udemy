1 - Bootstrap Intro

Notes:

Intro

  • It's a front end library
  • Front end - what the user sees, really pretty
  • Back end - normally on server, user doesn't see
  • Responsive means responsive to the viewport
  • Bootstrap has loads of pre-styled elements you can drop into your code
  • Go through the Bootstrap docs for info about elements you want to know about

CDN

  • Content Delivery Network
  • Means your files are installed many servers so it's quicker to load your site
  • E.g. if request starts from UK, it'll look for closest location
  • Dramatically reduces latency (how quick) website loads
  • Bootstrap files on maxcdn which has many locations
  • Because bootstrap common, it's likely that the user will already have these files on comp, further saving time

Installation

  • Go to boostrap installation for link to copy for your files
  • Can also use their starter template
  • Can also download CSS & JS source codes for your site but CDN recommended to reduce latency for now

Keywords/Questions:

  • Front end library
  • Can use Bootstrap for pre styled elements
  • It uses CDN and users normally have their files cached to reduce latency
