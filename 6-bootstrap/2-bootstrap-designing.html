Notes:

Wireframing

  • One of 1st jobs as web designer is to wireframe site, pencil & paper really basic
  • Mockup opposite to wireframe, high fidelity version of your site, looks exactly how you want it
  • Start by looking at competitors websites, ui-patterns.com to look at design patterns
  • Then do wireframes then optional next step is mockup & potentially prototype which has it's functionality in design
  • Dribble.com good site for inspo - designers put up their portfolios
  • sneakpeekit.com good for downloadable templates for wireframing plus balsamiq.cloud industry standard

Navbar

  • On Bootstrap docs, go to components, navbar
  • Use nav element
  • To see Bootstrap default colours see docs buttons
  • Can use all these colours easily by using the words needed
  • Bootstrap docs, utilities spacing to see how to move navbar to right
  • Many other similar frameworks to bootstrap - once you know bootstrap it's easy to pick up any others
